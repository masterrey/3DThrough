
Thank you for downloading 3dThrough!


This plugin is based on 3DConexxion device drive.
 
You can make profiles and change mouse configurations on 3DConexxion properties.

Please use stable version of 3DConexxion device drive only. 

For this plugin work, the 3dThrough window must be open at all the time.

You can use the brute axis value in game or other thing (E3dThrough.BruteTranslation). 

You can find it in "window/3dThrough/"

Buttons list: 



Button 2 alternate the view to the object movement. 

Button 3 its helicopter mode (the rotation origin is taken from the camera position and not from object origin)

Button 4 alternate orthogonal view;

Button 5 alternate bruteforce data exit; 

Buttons 7, 8, 9 and 10 are the usual view side modes.

Button 11 is the fit button, you can config this button in your mouse settings.



Troubleshooting:

If the program stops working, close the Unity editor and reopen it.


Eventualy 3DConexxion device drive doesn't start properly in your computer. 
If this happen, restart your drive before opening this plugin.
